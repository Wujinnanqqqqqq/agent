3. **RSI (14天)**
   - 平均增益 = 过去14天中上涨日的平均涨幅
   - 平均损失 = 过去14天中下跌日的平均跌幅
   - RS = 平均增益 / 平均损失
   - RSI = 100 - (100 / (1 + RS))
